# MOA
2022 하계 SJSU SVSTIP 해외교육에서 MashUP 팀이 진행한 프로젝트입니다. <br />
MOA는 <b>사용자의 취향을 리뷰와 콘텐츠 평가 데이터를 기반으로 분석해, 사용자 맞춤 OTT 플랫폼 및 콘텐츠를 추천해주는 애플리케이션📱</b>입니다. <br />
